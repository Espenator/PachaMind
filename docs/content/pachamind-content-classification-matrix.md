## How to use

Before any research, lesson, page, email, WhatsApp message, ad, or asset goes
public, classify it here. Columns: **Public (Flow A)**, **Login-only learning**,
**Counsel-only before use**, **Forbidden / never use**.

## Decision rules

1. If it teaches Andean cosmovision, culture, values, conservation, or Mallku
   attribution **without selling anything**, it can usually be public.
2. If it mentions Huachuma/San Pedro, ceremony, plant medicine, screened travel,
   paid membership, pricing, payment, booking, donations, or commercial contact,
   it is **not** public Flow A content.
3. Frame future revenue as **membership / membership levels**, never donations.
   Any donor, donation, or conservation-contribution funnel is forbidden in Flow A.
4. Communication permissions are **separate** (account messages, optional email,
   optional WhatsApp, optional later SMS). A free membership signup is **never**
   automatic Embodier.ai commercial consent. The risk is **not** collecting
   reasonable membership coordinates — it is using them for undisclosed
   commercial purposes before separate Flow B consent.
5. If it could affect nonprofit status, Ad Grants, SMS/WhatsApp consent, payment
   routing, operator liability, private benefit, data sharing, or Peru
   compliance, mark it `[NEEDS COUNSEL/VERIFICATION]`.
6. If it implies health, healing, treatment, cure, or guaranteed transformation,
   do not use it.

## Ad Grants conversion classification (Flow A)

**Count as conversions:** free PachaMind learning membership created, lesson
started, lesson completed, video watched, quiz completed, reflection submitted,
module completed, badge earned.

**Do not count:** paid membership interest, upgrade interest, Embodier.ai
commercial contact, screening interest, travel interest, Peru pathway interest.
