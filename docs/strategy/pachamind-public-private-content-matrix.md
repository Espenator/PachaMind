This matrix operationalizes the public/private content firewall for PachaMind.org and Embodier.ai. Use it before turning any research, lesson, ad, SMS, landing page, video, or lead magnet into a public asset.

## Decision rules

Use these rules when classifying new content:

1. If it helps a public visitor learn Andean cosmovision, culture, values, conservation, or Mallku attribution without selling anything, it can usually be public.
2. If it mentions Huachuma/San Pedro, ceremony, plant medicine, screened travel, paid membership, pricing, booking, or sales contact, it is not public Flow A content.
3. If it could affect nonprofit status, Ad Grants, SMS consent, payment routing, operator liability, private benefit, or Peru compliance, mark it [NEEDS COUNSEL/VERIFICATION].
4. If it implies health, healing, treatment, cure, trauma recovery, addiction support, anxiety/depression/PTSD outcomes, or guaranteed transformation, do not use it.
5. If it makes the Puma Path feel like a path to ceremony access or paid priority, rewrite it as learning-only.
