# PachaMind Strategy Canon

This directory is the durable GitHub home for PachaMind research and strategy
artifacts synced from the Perplexity PachaMind Space canon.

## Roadmap canon

The current sequencing is:

1. **Flow A** — PachaMind.org free, commercial-free education and Google Ad
   Grants readiness, first.
2. **Flow B** — Embodier.ai commercial SMS/CRM/membership bridge, second, only
   after counsel and a safe grant posture.
3. **Peru-only screened pathway** — third, after operational, legal, payment,
   and consent gates.

Entity separation between **PachaMind** (nonprofit education), **Embodier.ai**
(commercial bridge), **Mallku**, and Peru contractors is a hard guardrail
across all of these documents.
