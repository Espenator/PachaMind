# Homepage — AI-Assisted Build (Flow A)

## Overview
The PachaMind homepage was generated in Figma using the UX Pilot AI UI Generator plugin, then reconciled against the Flow A design canon. The frame lives on the **Mobile Screens** page as `Home (AI generated)`.

- Frame size: **390 x 2490** (mobile-first, single 390px column)
- Fills: surface `#F7F1E7` (warm parchment), card `#FFFFFF`
- Adheres to LOCKED CANON: free learning membership only — no pricing, currency, checkout, upgrade, donation, or booking language.

## Section map (top to bottom)
1. **Header** — PachaMind wordmark + hamburger menu.
2. **Hero** — Heading "Discover the wisdom of the Andes.", supporting copy, primary CTA "Start Learning Free", and the supporting line "Create free PachaMind learning membership".
3. **Featured lesson card** — video thumbnail with play affordance, label "The Concept of Ayni: Sacred Reciprocity".
4. **Why explore this path?** — three benefit cards: Earth Connection, Cosmic Harmony, Ancestral Wisdom.
5. **Latest Lessons** — section heading with "View all"; lesson rows: Weaving the Cosmos, The Chakana, K'intu & Offering; secondary CTA "Browse All Topics".
6. **Footer** — brand blurb, Explore + Platform link columns, social icons, copyright line.

## Canon compliance check
- [x] No pricing or currency references anywhere.
- [x] No checkout / upgrade / donation / booking UI.
- [x] CTAs use approved free-membership language ("Start Learning Free", "Create free PachaMind learning membership").
- [x] Mobile-first 390px frame.

## Tokens used
Colors map to the central `tokens.css` (additive). Surface and card fills reference the parchment/white palette already defined for Flow A. No new token files introduced.
