# Figma Flow A — Design Brief (Free Learning Membership)

## Locked Canon
PachaMind.org initial rollout offers FREE PachaMind learning memberships only. No paid tiers, no commerce, no booking, no ceremony pathways, no health/outcome claims.

## Flow A Purpose
Flow A is the free learning onboarding and engagement flow for PachaMind.org. It guides a visitor from the homepage into a free learning account (the Puma Path), where they can learn Andean cosmovision, save progress, complete reflections, and earn learning badges. The flow is educational and cultural-preservation focused.

## Membership Model
- Free PachaMind learning membership only.
- Free learning account that saves progress along the Puma Path.
- Primary CTAs: Start Learning Free; Create Free Membership; Continue the Puma Path.
- No pricing, checkout, payment, upgrade, donation, or contribution language anywhere.

## Visual Direction
Calm, credible, dignified, educational, documentary, earth-toned. Rooted in the Andes: stone, sky, woven texture, natural light. Warm editorial serif for headings, clean sans for UI.
Avoid: startup SaaS look, neon, purple/blue gradients, luxury retreat visuals, psychedelic visuals, urgency marketing, commercial landing-page patterns.

## Screen Inventory (mobile-first, 390px)
1. Home
2. Learn
3. Lesson Detail (The Three Pachas)
4. Puma Path
5. Signup (Create your free PachaMind learning membership)
6. Dashboard (mock logged-in)
7. Account Preferences
8. About Mallku
Optional: 1440px desktop Home adapting the mobile design with identical tone and free-membership-only language.

## Component Inventory
SiteHeader, MobileNav, SiteFooter, PrimaryButton (Start Learning Free), SecondaryButton (Explore Lessons), QuietLink (Continue), FeaturedVideoCard, LessonCard, BadgeCard, ReflectionCard, PumaPathStageCard, ProgressRing, TextInput, SelectInput, PhoneInput, ConsentCheckbox, LanguageToggle.

## Allowed Language
free PachaMind learning membership; free learning account; Start Learning Free; Create Free Membership; Continue the Puma Path; Save your progress; Complete reflections; Earn learning badges; Learn Andean cosmovision; Cultural preservation; Conservation education; Mallku Aribalo as tradition-holder and Andean shamanic teacher.

## Forbidden Language (must never appear)
paid memberships; paid upgrade CTAs; pricing; checkout; payment forms; fundraising; contribution language; donate/donor language; booking; applications; Peru travel offers; lodging/transport offers; ceremony pathways; Embodier.ai commercial opt-in; public Huachuma/San Pedro conversion hooks; psychedelic retreat language; health, healing, therapy, trauma, cure, addiction, anxiety, depression, PTSD, or outcome claims.

## Accessibility Notes
- Target WCAG AA contrast (>= 4.5:1 body text, >= 3:1 large text and UI).
- Visible focus ring on all interactive elements (terracotta outline).
- Tap targets >= 44x44px.
- Body text >= 16px; respect user font scaling.
- All inputs have associated labels; consent checkboxes are individually labeled and unchecked by default.
- Provide transcript for lesson video content.

## Mobile-First Requirements
- Design at 390px width first; scale up to 1440px.
- Single-column layouts, thumb-reachable primary actions, sticky MobileNav.
- Images use descriptive alt text; documentary, earth-toned imagery only.
